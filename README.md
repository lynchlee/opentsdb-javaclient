## OpenTSDB Java Client

#### What is this?

A simple java client used to read data from HBase table named `tsdb` of [OpenTSDB](https://github.com/OpenTSDB/opentsdb).

#### What does this do?

Query metrics datapoints from Storage "HBase" of OpenTSDB.
