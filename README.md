## OpenTSDB Java Client

#### What is this?

A simple java client used to read data from HBase `tsdb` table with [OpenTSDB](https://github.com/OpenTSDB/opentsdb).

#### What does this do?

Query metrics datapoints from HBase with components of OpenTSDB.
